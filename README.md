#### Union Avionics Simulator Center
项目简介：基于 Electron 开发的模拟器航电管理系统，实现硬件产品与多种模拟飞行平台的航空器实时数据交互，统筹模拟器航电产品固件更新、数据传输等核心功能，保障航电模拟精度与系统稳定性。

核心技术与成果：

- 数据交互架构：基于 Websocket 实现软件与模拟飞行平台的实时数据交互，解析航空器数据后生成增量数据包，通过 CCD 虚拟串口协议与模拟飞行硬件完成数据传输，保障数据实时性与准确性。

- 模块化与可维护性：抽象航电产品模块公共方法与接口，降低后续新产品开发与维护成本，提升开发效率 50% 以上。

- 固件升级保障：采用固件拆包策略，将二进制固件拆分为多个小数据包，通过自定义协议传输，同时硬件使用 RingBuffer 算法接收串口数据，避免硬件数据阻塞崩溃；追加 CRC 校验码，确保固件升级文件完整性，杜绝升级变砖问题。

- 源码安全防护：对 Electron 发布包源码进行字节码加密，有效防止源码泄露，保障核心技术安全。

- 技术栈：Node.js、 Vite、Websocket、Electron
